Case Study: MTs Al Marwah Website

This project is a company profile and admission system website for MTs Al Marwah. It includes features such as online registration (PPDB), selection results, printable forms, gallery, and an admin dashboard.

Project Overview

The goal was to build a modern, responsive school website with an integrated admission system that simplifies registration for new students and provides essential information for parents and the community.
